Linking The Style Sheet

You have to link a stylesheet to an HTML document using the (link tag)

  • href = The location of the target you are linking, is actually called the anchor element.
  • type = type of document you are attempting to link to. Value should be set to text/css
  • rel = defines the relationship between the documents you are linking. Should be set to stylesheet.

Example: (link href= "C:\Kyle\School\Sec 4\Computer Programming\Css for Notes 2.css" type= "text/css" rel= "stylesheet"/)
The example is also the stylesheet that is affecting this code.

Modifiers

You can put modifiers, direcly in the HTML code or on the stylesheet. Use the attribrute style or the style tag in the header to set the style in the (head). Example
(p style="color: red;")Red 2, The Movie (/p)

This gives:

Red 2, The Movie

In the style sheet, declare the target, then declare the attributes and their values between {} and ending them with End style with ;

You can use color... to change the color.

Use font-size to set the font size. Add a value in pixels. For example:

20px

Use font-family to set the font type. Set the value to the name of a font family, like Arial.

In order to set the text to bold use the font-weight element. Set the values to bold or normal to do this.

Use the text-align element to align the text to the left, right or center.

Use background-color to set the background color for something, which is basically CSS highlighting.

Use opacity to set the transparancy of an element. 0 = 0% transparency and you can go all the way up to 1 which equals 100% transparency.

background-image makes the background the target element a target image. The image should be within brackets and in quoatations next to a declaration of url.. Example: url("https://vignette.wikia.nocookie.net/hdm/images/7/7e/Modern_World_Golden_Compass.jpg/revision/latest?cb=20131108101726")

Target

Class can be used to target multiple or just one element of the code. Setting something to the class "Theodore " and then if you change something that affects this class, this element will change to.

For example, changing the color of the class ppiinnereaapppppllee will change color of the word ppiinnereaapppppllee and Theodore .

Put . in front of your class when you call it in your Css style sheet.

You may also use id to target something.

Simply use a # instead of a . in the stylesheet.

Example of calling a class: (p class= "BigBoy") 20px (/p)

If the you wish to call more then one element for a rule on the stylesheet, put a comma between them.

Not putting a comma, makes the element nested. Nested element make the rule's target more specific as the target element must correspond to all the elements presented.

Adiding !important will make that rule overide anything.

The Box Model

Border is the border, the line, the frontier, of the box model.

Modify border corners with border-radius and put in a pixel value. Does not need to be a sqaure or a rectangle, to make the border a circle declare a value of 100%.

Padding: "space between a picture and a frame" (Direct quote from Code Academy, and most suff here is taken from it.

Use padding to set the padding space. Put a pixel value. You can use this to extend background color and suff and make stuff less stuck together.

You can more specific with each padding-(side) you want bigger. Example: top, left, right, bottom or put multiple values in one statment.
Values will be assigned in this order: (top)px, (right)px, (bottom)px, (left)px. Only two values makesthe first one equal to the top and bottom and the second one uses the left and right side padding width.

Margin refers to tbe space "outside the box". And follows the same rues as padding.

MAKE YOUR LIFE EASIER (maybe) WITH AUTO! Putting the auto makes the margins change until the elements is centered with all the other ones.

Makes sure to give an element a width and a height for you can not center something that takes up the whole screen.

Use width and height to set the width and height.

When two margins are next to each other they are added together forever in a bond of beautiful union and are forever displayed as such. Therefore two 10px margins would add to 20px.

Webpage are always changing because of computer multi culturism so you can use max-width and min-width to save yourself from being assimilated. Height can also be protected. Be careful, assimilation can cause text spillage if you make these values baddly.

In case of overflow use overflow to unoverflow the overflow with these handy properties:

  • hiddej: hides the overflow, scroll: adds a scroll bar so you can scroll down it
  • visible: makes the suff visible and is what is it is set to if you do not put anyhting else.

In order to reset the defalt style sheets use:

				margin: 0;

  			}

Changing the box model

The tag property box-sizing is used to define what box model to use. The box model is normally set to content-box, but, you can also set it to border-box.
This property is put in between the braquets refering to the * propety, so * {}.
* is the... is the... universal selector.

In the actual box model the total width is all the property combined inclding the set content width, but, in the new box model the content width is set to the remaining width on the page.

Position

Use the position property to set the position. You can set it to:

  • static which is the default and just where you put it in your html code.
  • relative lets you put in it relative to its static location in the html code by pixel, em or percentage values in the top (move down) bottom (move up) left (move right) left (move right) properties.
  • absolute makes everybody pretend it that the element associated with the code does not exist and that the element will be displaced and if you recise it it will cover things and stuff.
  • fixed makes something stay in place no matter what a person does, no scrlling escapes, but you must define it like the relative property.

z-index: What goes on top of what. The bigger the associated value, the more on top that element is!

display: can it share horizontal space, or what? Does not make it bigger, or does it make the element next to it skip a line? So many questions, and so much more code to write.

  • inline makes a box only take the room it needs, shares lines.
  • When display is set to block the element fills up the width of the page and takes the height it needs.
  • Setting display to inline-block means you can set it using height and width.
  • Float makes it move in a direction to its max, use the values left and right. Elements can be block each other if ther are multiple floaters so use clear to regulate this. The value left means that element will not touch any element within its conatiner on its left side, same for right and both and none is just none.
